# 프로젝트 투입 인력 및 부하율 타임라인

프로젝트별/인원별 인력 투입 및 부하율을 시각화하는 웹 애플리케이션입니다. 일/주/월/분기 뷰와 년 단위 네비게이션을 지원합니다.

## 개요

- **프로젝트별 뷰**: 프로젝트를 행으로 표시, 각 프로젝트에 투입된 인원의 할당 바 표시
- **인원별 뷰**: 구성원을 행으로 표시, 각 인원의 프로젝트별 할당 바 표시
- **타임라인 뷰**: 일/주/월/분기 스케일 전환, 년 단위 이동 (1년씩), 오늘로 이동, 스크롤 시 월/기간 헤더 상단 고정
- **드래그앤드롭**: 프로젝트별 뷰에서는 구성원 칩을, 인원별 뷰에서는 프로젝트 이름을 타임라인에 드롭하여 할당 생성
- **할당 바 드래그/리사이즈**: 드롭 직후 스토어 반영 전에도 위치·너비가 한 번에 유지되도록 처리(깜빡임·원래 자리로 잠깐 돌아가는 현상 완화). 날짜·세그먼트 갱신은 Zustand에서 한 번에 반영
- **부하 시각화**: 주차별 총 투입 공수에 따라 색상 표시 (1.0 이하: 초록, 1.5 이하: 노랑, 그 이상: 빨강)
- **사이드바 필터**: 좌측 구성원 칩·프로젝트 항목을 클릭해 타임라인에 해당 항목만 표시(다중 선택·재클릭 해제, 미선택=전체 표시). 구성원 영역에 **선택 해제**로 필터 초기화
- **인원별 총계 필터 연동**: 필터 적용 시 선택된 인원/프로젝트 기준으로 해당 직군만 총계에 표시하고, 제목을 `[직군] 직군 총계` 또는 `선택 직군 총계`로 변경
- **데이터 내보내기/불러오기**: JSON 파일 및 URL 공유 지원

## 기술 스택

- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- date-fns (날짜 처리)
- (선택) Supabase: Postgres + Auth (공개 읽기, 로그인 후 쓰기)

## Supabase 연동 (선택)

`.env.local`에 다음을 넣으면 데이터가 Supabase에 저장되고, 비로그인 사용자는 읽기만 할 수 있습니다.

```bash
cp .env.example .env.local
# VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY 입력
```

1. [Supabase](https://supabase.com)에서 프로젝트 생성  
2. **SQL Editor**에서 `docs/supabase/schema.sql` 전체 실행 (테이블 + RLS)  
3. **Authentication → Providers**에서 Email 활성화 (필요 시 이메일 확인 비활성화는 프로젝트 설정에서)  
4. **Settings → API**에서 Project URL / anon public key를 `.env.local`에 복사  
5. `npm run dev` 재시작  

환경 변수가 없으면 이전과 동일하게 **localStorage**에만 저장됩니다.

## 배포 (Vercel 등)

호스팅 서비스의 **Environment Variables**에 로컬 `.env.local`과 동일하게 `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`를 넣은 뒤 빌드·배포합니다. 배포 후 Supabase **Authentication → URL Configuration**에서 프로덕션 도메인을 허용할 수 있습니다(이메일 링크·OAuth 사용 시).

## 주요 기능

- 프로젝트 생성/삭제 (기본 1개월 기간)
- 구성원 등록/삭제 (이름, 직군: FE/BE/기획/UI디자인/DS/PM/BD/운영)
- 타임라인 년 이동 (이전/다음 년도), 오늘 버튼 (다른 년도에서도 현재 년·오늘로 이동)
- 프로젝트 시작/종료 마커 드래그로 날짜 조정
- 할당 바: 전체 드래그(이동), 양 끝 드래그(기간 조정), 클릭(비율/역할 설정); 드래그 종료 시 시각적 적용이 끊기지 않도록 pending 레이아웃 유지
- 투입 비율: 0.01~1.0, 소수 둘째자리
- localStorage 자동 저장 (Supabase 미사용 시) / Supabase 동기화 (설정 시)
- 로그인 후 구성원·프로젝트·투입 수정 (Supabase + RLS)
- JSON 내보내기/불러오기/URL 복사 공유
- 좌측 구성원·프로젝트 다중 필터(선택 시 칩·행 강조, 할당이 없는 조합은 행 숨김)
