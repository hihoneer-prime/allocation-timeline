# 프로젝트 투입 인력 및 부하율 타임라인

프로젝트별/인원별 인력 투입 및 부하율을 시각화하는 웹 애플리케이션입니다. 일/주/월/분기 뷰와 년 단위 네비게이션을 지원합니다.

## 개요

- **프로젝트별 뷰**: 프로젝트를 행으로 표시, 각 프로젝트에 투입된 인원의 할당 바 표시
- **인원별 뷰**: 구성원을 행으로 표시, 각 인원의 프로젝트별 할당 바 표시
- **타임라인 뷰**: 일/주/월/분기 스케일 전환, 년 단위 이동 (1년씩), 오늘로 이동
- **드래그앤드롭**: 좌측 구성원 칩을 타임라인에 드롭하여 할당 생성
- **부하 시각화**: 주차별 총 투입 공수에 따라 색상 표시 (1.0 이하: 초록, 1.5 이하: 노랑, 그 이상: 빨강)
- **데이터 내보내기/불러오기**: JSON 파일 및 URL 공유 지원

## 기술 스택

- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- date-fns (날짜 처리)

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 주요 기능

- 프로젝트 생성/삭제 (기본 1개월 기간)
- 구성원 등록/삭제 (이름, 직군)
- 타임라인 년 이동 (이전/다음 년도), 오늘 버튼 (다른 년도에서도 현재 년·오늘로 이동)
- 프로젝트 시작/종료 마커 드래그로 날짜 조정
- 할당 바: 전체 드래그(이동), 양 끝 드래그(기간 조정), 클릭(비율/역할 설정)
- 투입 비율: 0.01~1.0, 소수 둘째자리
- localStorage 자동 저장
- JSON 내보내기/불러오기/URL 복사 공유
